开练题库项目

本文最后更新于 2025年2月24日 晚上

一、系统功能列表

  1. 用户体系
  • 注册/登录（手机号+密码）
  • JWT/SA-Token鉴权
  • 角色权限管理（普通用户、管理员）
  • 个人信息管理
  1. 题库管理
  • 多级学科分类（小学/初中/高中/大学→数学/语文→章节）
  • 题目类型管理（单选/多选/填空/判断）
  • 题目难度分级（简单/中等/困难）
  • 题目增删改查（支持富文本和公式）
  • 批量导入导出（Excel模板）
  1. 练习功能
  • 智能组卷（根据分类/难度随机抽题）
  • 答题计时与保存进度
  • 自动判卷与得分统计
  • 错题自动归集
  • 历史练习记录查询
  1. 数据统计
  • 个人能力分析雷达图
  • 正确率趋势折线图
  • 错题知识点分布
  1. 管理功能
  • 用户权限管理
  • 系统访问日志
  • 题目审核流程

二、技术栈组合

  1. 后端
  • Spring Boot 3.2 + MyBatis Plus 3.5.6
  • Sa-Token 1.37 + JWT
  • Maven 3.9 + Java 17
  • MySQL 8.0 + Redis 7.0
  1. 前端
  • Vue3 + TypeScript
  • Vite 5.0
  • Element Plus 2.4
  • Axios 1.6
  • ECharts 5.4

（包含完整的权限管理、题目管理、练习模块和可视化统计功能）

部署步骤：

  1. 创建MySQL数据库并执行提供的schema.sql
  2. 修改application.yml中的数据库配置
  3. mvn clean package 打包后端
  4. npm install && npm run build 构建前端
  5. 启动Redis服务
  6. 运行后端jar包：java -jar exam-bank.jar

系统特色：

  1. 采用RBAC权限模型实现细粒度控制
  2. 使用Redis缓存高频访问的题目数据
  3. 支持LaTeX数学公式渲染
  4. 自动生成智能练习报告
  5. 完整的数据埋点与分析功能
