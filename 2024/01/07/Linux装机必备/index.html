由于出现了 “invalid packet” 的错误消息，可能是由于密钥文件本身的问题导致的。建议您尝试使用以下方法来解决该问题：

  1. 删除 /etc/apt/trusted.gpg 文件：

    sudo rm /etc/apt/trusted.gpg

  2. 创建一个新的空白 trusted.gpg 文件：

    sudo touch /etc/apt/trusted.gpg

  3. 导入密钥：

    sudo apt-key adv –keyserver keyserver.ubuntu.com –recv-keys

    注意，将 <KEY> 替换为您要导入的密钥的指纹。

  4. 重复步骤3，为每个要导入的密钥重复该步骤。

2、安装MySQL8.x

安装已下载的deb包

即出现黄色界面，因为deepinV20是基于Debian10,因此我们选择第三项”Debian buster“

然后选择确认

确认后出现版本选择界面，选择第一项，并确认

确认后会出现警告报错。不用担心，看下一步。

接下来执行：sudo apt-get install mysql-server

选择【Y/N】输入Y

如果不报错，请略过这一步

如果出现这样的报错，执行：sudo apt-get update –fix-missing

完成后，再次执行：sudo apt-get install mysql-server

同样选择Y。

等待下载

再次出现黄色界面时，说明成功一半了。

输入root 密码 确认

再次输入密码 确认

点击退出即可

点击确认即可

等待安装即可

到这说明安装完了

确认下MySQL服务器版本，执行：apt policy mysql-server

大功快告成！

启动MySQL服务，执行：sudo systemctl start mysql

再执行进入命令：mysql -u root -p
